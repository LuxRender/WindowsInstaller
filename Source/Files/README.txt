LuxRender pre-0.9
=================

LuxRender is a physically based and unbiased rendering engine. Based on
state of the art algorithms, LuxRender simulates the flow of light
according to physical equations, thus producing realistic images of
photographic quality.


What's new in 0.9:
------------------
- Normal mapping support
- Multi-GPU support for Hybrid renderer
- Improved network error handling
- Layered material
- Glossycoating material
- Fresnelcolor and fresnelname fresnel textures
- Improved motion-blur support, allowing for complex motion paths
- Support for adding files to Render Queue from command line


What's new in 0.8:
------------------
 - Volumetric Scattering
 - Ex. Photon Map fixed and improved
 - Outlier Rejection
 - Glossy Translucent material
 - Texturing Improvements
 - Film Response functions
 - Microdisplacement
 - PLY/STL Mesh Loading
 - GPU Acceleration
 - Core Improvements
 - GUI Improvements
(see http://www.luxrender.net/wiki/New_in_0-8 for details)
